# 🤖 Gmail 助理機器人

自動幫你**讀信 → 分門別類 → 備回覆草稿 → 每日三時段統整報告**。
獨立 Python 程式，用**免費 Gemini API**，靠 Windows 工作排程器定時跑，不消耗任何付費額度。

專為成大醫院心臟內科醫師的信件型態設計（期刊、學會、醫院公布欄、薪資、消費通知…）。

## 它會做什麼
- **15 類自動分類**，加上 `🤖/` 系列 Gmail 標籤（與你手動標籤互不干擾）
- **只幫需回覆的真人信備草稿**（放草稿匣，絕不自動寄出）
- **三時段報告**寄給自己：
  - `08:00` 每日完整統整（昨 0800 → 今 0800）
  - `17:00` 白天進度快報
  - `22:30` 當日收尾 ＋ 明天注意
- **緊急信即時提醒**、**待辦/Deadline 抽取**、**跟催提醒**（你寄出對方沒回）、**長信 TL;DR**

## 安裝
看 **[SETUP.md](SETUP.md)**（照做一次，約 20 分鐘）。

## 檔案結構
| 檔案 | 作用 |
|---|---|
| `main.py` | 主程式 / 進入點（process / morning / afternoon / night / test） |
| `config.py` | 15 類定義、規則式寄件者對應、參數 |
| `gmail_client.py` | Gmail API：授權、抓信、標籤、草稿、寄信、跟催 |
| `gemini_client.py` | 免費 Gemini：分類、摘要、起草回覆 |
| `classifier.py` | 混合分類：規則優先（省額度）、其餘交 Gemini |
| `reporter.py` | 組裝三時段 HTML 報告 |
| `database.py` | SQLite 狀態（避免重複處理/重複草稿） |
| `register_tasks.ps1` | 一鍵註冊 Windows 排程 |
| `.env` | 你的金鑰與參數（由 `.env.example` 複製） |

## 設計重點
- **混合分類省額度**：信用卡通知、促銷、電子報等用寄件者規則直接歸類，不呼叫 Gemini；只有重要/模糊的信才動用 AI。每天用量遠低於免費上限。
- **冪等**：處理過的信記在 SQLite，重複跑不會重做或重複建草稿。
- **安全**：草稿一律人工過目才送；緊急提醒可在 `.env` 關閉。
